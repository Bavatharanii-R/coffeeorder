# ☕ Ctrl + Al‑latte: Coffee Order App

A fun and simple full‑stack project built with **Spring Boot (Java backend)** and **HTML/CSS/JavaScript frontend**.  
It lets users place coffee orders, assigns each order an auto‑generated ID, and displays them neatly in a styled table.

---

## 🚀 Features
- Place coffee orders with customer name and coffee type.
- Auto‑generated order IDs for tracking.
- Orders displayed in a clean, styled table.
- REST API endpoints tested with Postman.

---

## 🔧 How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/Bavatharani-R/coffeeorder.git
2. Navigate into the project:

```bash
  cd coffeeorder
```
3. Run the backend (Spring Boot):

```bash
  mvn spring-boot:run
```
4. Open index.html in your browser (or place it in src/main/resources/static).
